# 🍃Leaf-Type-Prediction-Project-Betel-or-Black-pepper

This project uses machine learning techniques to classify betel and black-paper leaves based on their features. The model is trained on a labeled dataset of leaf characteristics to accurately predict the leaf type, demonstrating the application of data preprocessing, feature extraction, model training, and evaluation in a real-world classification problem.
